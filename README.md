# githubclient

Implemented by:

1) Displaying a list of Github users:
- user list retrieval screen - GetUsersActivity
- in the list item are drawn avatar, login, id
- by clicking on the list item, the transition to the detailed information about the user is implemented.

2) Display User Details. UserDetailsActivity screen with user information:
- Avatar
- Name
- Email
- Organization
- Following count
- Followers count
- Date of account creation.

3) Work with the network. API is used https://api.github.com/

4) Pagination list

5) The architecture of the application Android architecture components
